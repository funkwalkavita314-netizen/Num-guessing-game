# Num-guessing game

# Project Description
This is a simple Python-based Number Guessing Game where the user tries to guess a randomly generated number within a given range.
The program provides hints such as whether the guessed number is too high or too low until the correct number is guessed.


# Why I Created This Project
I created this project to practice:
1. Python basics
2. Loops(while loop)
3. Conditional statements(if-else)
4. Random number generation
5. User input handling
6. Logical thinking

This project helped me understand how game logic works and how to interact with users through the terminal.


# Features
1. Random number generation
2. User input validation
3. Hint system
4. Score or attempt tracking
5. Simple and beginner-friendly logic.


# How to Run
1. Download or close the repository
2. Open terminal in the project folder
3. Run the following command:
   
python main.py
